# FirstApp - Interface Netflix

Este projeto é uma atividade prática da disciplina **Programação para Dispositivos Móveis**, com o objetivo de criar uma interface semelhante à do aplicativo Netflix.

## ✅ Funcionalidades
- Tela principal com:
  - Logo Netflix
  - Banner principal
  - Botões **Play** e **Info**
- Tela de detalhes com:
  - Banner
  - Título e descrição do filme

## ✅ Tecnologias
- Android Studio
- Linguagem: Java
- Layout: XML com **ConstraintLayout**

## ✅ Como executar
1. Abra o projeto no Android Studio.
2. Conecte um dispositivo Android com **Depuração USB** ou use um emulador.
3. Clique em **Run** para executar.

## ✅ Estrutura
- `MainActivity.java` → Tela principal
- `DetailsActivity.java` → Tela de detalhes
- `activity_main.xml` → Layout principal
- `activity_details.xml` → Layout detalhes
- Imagens → Pasta `res/drawable`

## ✅ Observação
Este projeto é apenas para fins educacionais, explorando elementos de interface gráfica no Android.
